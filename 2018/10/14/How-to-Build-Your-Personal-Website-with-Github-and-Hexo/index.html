What is Hexo?

​ Before we started, I’m going to talk something about Hexo. Hexo is a static blog framework which based on Node.js, it has no so much depend on and very convenient for useing. It can generate static web pages and host them on Github or Heroku. meanwhile, Hexo is also a open source project on Github – hexojs/hexo. If you want to learn more about Hexo, you can go to the official website of Hexo.As for Github, I think you can no longer be familiar with it, so I will not talk too much on this part, but if you want to learn more about Git, I recommend this aitical – 《Git使用教程》.
